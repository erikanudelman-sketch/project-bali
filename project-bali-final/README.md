# Project Bali

A mobile-first 50-day fitness and wellness tracker.

## Data and privacy

Tracking entries are stored in the browser's local storage on the device where the app is used. They are not committed to GitHub and are not sent to a database. Export a backup regularly from the app. A basic Vercel URL is accessible to anyone who knows it, although this project asks search engines not to index it.

## Deploy with GitHub and Vercel

1. Upload all files and folders in this project to the root of your `project-bali` GitHub repository.
2. Sign in to Vercel with GitHub.
3. Import the `project-bali` repository.
4. Vercel should detect Vite automatically. Keep the default build settings and deploy.
5. Open the Vercel URL in Safari on iPhone.
6. Tap Share, then Add to Home Screen.

## Local development

```bash
npm run dev
```
